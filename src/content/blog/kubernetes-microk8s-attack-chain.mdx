This is a full chain I documented from an external assessment where Kubernetes ports were internet-facing. It goes from Nmap to root on the host running MicroK8s.

```bash

```

## Lessons

1. **10255 should not face the internet.** Anonymous pod enumeration starts the chain.
2. **Credentials on web-accessible paths** (`/.git-credentials`) still happen.
3. **MicroK8s on SSH-accessible hosts** with permissive pod security = host compromise.
4. Patch MicroK8s and enforce Pod Security Standards.
